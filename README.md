# Real-Time Cafe Automation System with Admin Dashboard

## 📌 Overview
This project is a full-stack cafe automation system designed to streamline cafe operations and improve customer experience. It allows users to browse the menu, place orders online, and book tables, while providing admins with full control over the system.

## 🚀 Features

### 👤 User Module
- View categorized menu items
- Add items to cart and place orders
- Book tables online
- Responsive and user-friendly interface

### 🔐 Admin Module
- Admin dashboard for full control
- Add, update, and delete menu items
- View and manage customer orders in real-time
- Update order status dynamically

## ⚙️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python
- Database: JSON-based file storage

## 🔄 System Functionality
- Real-time order updates without page reload
- Dynamic menu rendering from JSON data
- Admin-side control panel for complete system management
